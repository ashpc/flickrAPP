Welcome to the flickr API front end exercise. In this exercise, I have showcased making calls to get the public photos and displaying them with images along with details that can be viewed upon clicking.
Some of the features included for the current page

  • Api Based Results -- By default Recent Photos Api is used
  • Sorting -- By default Sorted by ID
  • Filter Sizes -- By default set to square
  • Search
  • Pagination
I have used the following to display the usage of the flickr API's.
HTML5
AngularJS -- For the front end MVC
Bootstrap -- For RWD
FontAwesome -- For font icons to be used instead of images
